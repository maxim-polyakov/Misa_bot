# Очистка S3 от лишних изображений

## Пометка изображений

- **Telegram/Discord** — сохраняются в `images/telegram/` и `images/discord/` (помечены для очистки)
- **Веб-чат** — сохраняются в `images/` (не удаляются автоматически)

## Регулярная очистка (скрипт)

Скрипт удаляет **помеченные** изображения (telegram/discord) старше 7 дней.

**Запуск вручную:**
```bash
cd server
python scripts/cleanup_s3.py
```

**Cron (ежедневно в 3:00):**
```cron
0 3 * * * cd /path/to/Misa_bot/server && python scripts/cleanup_s3.py
```

**Срок хранения** — переменная `S3_CLEANUP_DAYS` в `.env` (по умолчанию 7).

## Текущее поведение

- **Очистка чата / удаление чата** — веб-изображения сразу удаляются из S3
- **Скрипт cleanup_s3.py** — удаляет telegram/discord изображения старше N дней
